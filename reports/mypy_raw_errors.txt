## Most Affected Directories

1. **CommonStructure/** - Timing constraints, mode declarations, resource consumption
2. **SystemTemplate/Fibex/** - Core communication, multiplatform fibex
3. **MSR/DataDictionary/** - System constants, record layout, units
4. **SWComponentTemplate/** - End-to-end protection
5. **ECUC** - Description/Parameter templates

## Key Patterns

1. **Filter functions** - Missing TypeGuard annotations
2. **Optional returns** - Should be `T | None` not just `T`
3. **List append** - Type mismatch between expected and actual element types
4. **Assignment from None** - Optional types not properly annotated
5. **Missing modules** - Unimplemented AUTOSAR classes (MeasurementCalibration, SignalServiceTranslation)
